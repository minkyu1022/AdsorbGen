# Experiments

This directory holds reproducibility scripts for specific research episodes.
These files are not part of the active AdsorbGen library API.

## Layout

- `2026-05-oc20dense/`: OC20-Dense MLIP relaxation and slab-reference materialization.
- `2026-05-mlip-pass/`: AdsorbSample-style OOD-50 MLIP Pass@k sanity check.

Prefer adding future one-off research code here instead of under
`adsorbgen/scripts/` or root `scripts/`.
